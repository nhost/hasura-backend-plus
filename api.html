# API

Category Path Description
Authentication POST /auth/register Account registration
POST /auth/login Login
GET /auth/jwks JWK Set
POST /auth/account/activate Activate account
POST /auth/account/delete Delete account
POST /auth/password/forgot Forgotten password
POST /auth/password/reset Reset password
POST /auth/email/forgot
POST /auth/email/reset
POST /auth/token/refresh Refresh token
POST /auth/token/revoke Revoke token
POST /auth/mfa/generate Generate MFA QR code
POST /auth/mfa/enable Enable MFA
POST /auth/mfa/disable Disable MFA
POST /auth/mfa/totp TOTP
Storage GET /storage/
POST /storage/
DELETE /storage/
GET /storage/meta/
POST /storage/meta/
DELETE /storage/meta/
Other GET /healthz Health Check
